# Public Assets Directory

Place your images and static assets here:

- `/images/` - Portfolio and gallery images
- `/assets/` - Icons, logos, and other media

## Recommended Image Sizes

- Hero image: 1200x800px
- Gallery thumbnails: 600x600px
- Profile image: 400x400px
- Service images: 640x480px

## Image Naming Convention

Use descriptive, lowercase filenames with hyphens:
- `hero-banner.jpg`
- `bridal-makeup-1.jpg`
- `profile-harshitha.jpg`
- `service-party-makeup.jpg`

All images should be optimized for web (compressed but maintaining quality).
